# GLM-4.5: Agentic, Reasoning, and Coding (ARC) Foundation Models

## 原标题 + 中文标题
**GLM-4.5: Agentic, Reasoning, and Coding (ARC) Foundation Models**
**GLM-4.5：智能体、推理与编码（ARC）基座模型**

## 一句话摘要
智谱 AI 发布 GLM-4.5——首个在单一模型中实现推理、编码和智能体能力原生融合的开源 MoE 基座模型，总参数 355B / 激活参数 32B，在 23T token 上进行多阶段训练，12 项全球核心基准中综合排名全球第三、中国第一。

---

## 完整核心内容翻译

### 发布背景

2025 年 7 月 28 日，智谱 AI 正式发布新一代旗舰模型 GLM-4.5。这是一款专为智能体应用打造的基座模型，首次突破性地在单一模型中实现了推理（Reasoning）、编码（Coding）和智能体（Agentic）三大核心能力的原生融合——智谱将这一能力组合命名为 **ARC**。

模型以 MIT 许可证在 Hugging Face 和 ModelScope 上同步开源，面向商业使用完全开放。

### 架构设计

GLM-4.5 采用稀疏激活的**混合专家（MoE）**架构：

| 版本 | 总参数量 | 激活参数量 |
|------|---------|-----------|
| GLM-4.5 | **355B** | **32B** |
| GLM-4.5-Air | 106B | 12B |

架构设计的一个关键决策是**优先加深而非加宽**——研究发现更深的模型具有更好的推理能力。模型使用 **96 个注意力头** 每层，并采用了以下关键技术组件：
- **QK-Norm**：对注意力机制中的 Query 和 Key 进行归一化，提升训练稳定性
- **Grouped Query Attention（GQA）**：分组查询注意力，优化推理效率
- **Multi-Token Prediction（MTP）**：多 token 预测，加速训练和推理
- **Muon 优化器**：改进的优化算法，实现更快收敛和更好的推理性能

### 训练策略

GLM-4.5 采用**两阶段预训练**策略，总计约 **23 万亿 token**：

1. **通用预训练阶段**：在 **15T token** 的通用语料上进行训练，建立广泛的世界知识和语言理解能力
2. **领域强化预训练阶段**：在 **7T token** 的代码与推理专项语料上进行训练，深化编程和逻辑推理能力

### Slime 强化学习框架

智谱为 GLM-4.5 的后训练阶段开发了开源的 **Slime** 强化学习框架，其核心特性包括：

- **灵活的混合训练模式**：支持同步和异步两种训练模式的灵活切换
- **智能体导向的架构设计**：将部署与训练解耦，便于在智能体场景中进行迭代优化
- **混合精度加速**：使用 FP8 进行推理生成，BF16 进行训练更新，兼顾速度与精度

### 基准测试表现

GLM-4.5 在 12 项全球公认的核心基准中取得了综合**全球第三、中国第一**的成绩：

| 基准测试 | GLM-4.5 成绩 | 说明 |
|---------|-------------|------|
| AIME 2024 | **91.0%** | 数学竞赛推理 |
| SWE-bench Verified | **64.2%** | 软件工程任务 |
| TAU-Bench | **70.1%** | 智能体基准（排名第二） |
| MMLU Pro | 高分 | 综合知识理解 |
| MATH 500 | 高分 | 数学推理 |
| LiveCodeBench | 高分 | 实时编程能力 |
| SciCode | 高分 | 科学编程 |
| GPQA | 高分 | 研究生级科学问答 |
| HLE | 高分 | 高难度语言评估 |

**视觉版本 GLM-4.5V** 更是刷新了 **41 项多模态推理 SOTA**（State-of-the-Art）。

### 定价

- 输入：**0.8 元/百万 token**
- 输出：**2 元/百万 token**
- 生成速度：最高 **100 tokens/秒**

### 后续发展：GLM-5

智谱已于 2026 年初发布 GLM-5，这是一款专为复杂系统工程与长程 Agent 任务设计的新一代旗舰模型，在后端架构设计、复杂算法实现及顽固 Bug 修复上展现出卓越的深度推理能力。

## 技术要点

1. **ARC 三能力原生融合**：与在通用模型上"外挂"智能体能力不同，GLM-4.5 从预训练阶段就将智能体、推理和编码三大能力作为共同优化目标，实现了能力之间的深层协同而非简单叠加。

2. **"深而非宽"的 MoE 设计哲学**：智谱的研究发现，增加模型深度比增加宽度对推理能力的提升更为显著。这一反直觉的发现挑战了"更多专家 = 更强能力"的朴素假设。

3. **23T token 两阶段训练**：15T 通用 + 7T 代码/推理的训练策略，精确控制了"通用知识"与"专项能力"之间的比例，避免了"专项能力强但常识差"的常见问题。

4. **Slime 框架开源**：作为中国 AI 公司中少数开源 RL 训练框架的举措，Slime 不仅服务于 GLM-4.5 的训练，更为整个开源社区提供了可复现的后训练基础设施。

5. **355B/32B 的参数效率比**：约 9:1 的参数效率比，在保持大规模知识储备的同时实现了高效推理，使 GLM-4.5 在成本效率上具有显著优势。

## 深度解读

GLM-4.5 的发布代表了智谱 AI 在技术路线上的一次重要转向：从"追赶通用 AGI"到"深耕智能体基座"。

**"ARC"概念的提出具有前瞻性。** 在 2025-2026 年 AI 行业集体转向 Agent 应用的大背景下，智谱率先提出将"智能体能力"与"推理"和"编码"并列为基座模型的核心能力维度。这不仅是一种学术贡献，更是一种产品定位的战略选择——GLM-4.5 不是要做"最聪明的模型"，而是要做"最好用的 Agent 底座"。

**TAU-Bench 70.1% 的成绩值得特别关注。** TAU-Bench 测试的是模型在真实智能体场景中的表现——包括工具使用、多步规划、错误恢复等能力。全球第二的排名表明，GLM-4.5 在智能体应用中的实际表现可能优于其在传统基准上的排名。

**开源 Slime 框架的战略意义。** 在 RL 训练框架成为 AI 公司核心竞争力的当下，智谱选择开源 Slime，既是技术自信的体现，也是"以框架建生态"的战略选择。如果更多团队基于 Slime 进行后训练，GLM 系列模型就会成为事实上的"训练标准参考"。

**从 GLM-4.5 到 GLM-5 的快速迭代**，则显示智谱已经建立了一套高效的"预训练-后训练-产品化"流水线。两代旗舰模型之间的间隔可能仅有半年左右，这一速度在全球范围内都是第一梯队。
