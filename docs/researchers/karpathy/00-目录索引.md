## Karpathy 的核心思想线索

纵观这 7 篇文章，可以提炼出 Karpathy 十年思考的几条主线：

1. **简洁性崇拜**：从 char-rnn 到 microgpt，他不断证明复杂系统的核心算法惊人地简洁
2. **理解优先于使用**：从"理解反向传播"到"训练秘诀"，他始终强调对底层机制的理解比工具使用更重要
3. **规模是主要变量**：从"33年"的分析中，他得出算法本质未变、规模驱动进步的结论
4. **范式思维**：从 Software 2.0 到 vibe coding，他擅长识别和命名正在发生的范式转换
